# Creamlon manifest

This repository publishes Creamlon protocol files through the bundled layout.

- Manifest: `.creamlon/manifest.yaml`
- Public trust records: `.creamlon/trust/`
- GitHub transport: Issues on this repository

The `x402-analysis-report` capability sells one-time credentials through x402
payment hints and uses HPKE private delivery for markdown input and PDF output.

Private `.creamlon` state such as keys, credentials, delivery outboxes, and
caches must remain local and ignored by exact path.
